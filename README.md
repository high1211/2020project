## 功能性需求:
1.遊戲規則簡單明瞭
2.豎立交通安全觀念

## 非功能需求:
1.適合全家大小遊玩
2.可靠性:遊戲規則遵循交通法規
3.遊戲界面簡易，好操作
4.系統管理者方便管理

## 需求分析:
1.遊戲流暢度高
2.遊玩過程有清楚明確的交通告示
3.可以駕駛不同車種
